# animationProgramming
本项目是个人制作的一些动画编程的demo，涉及三角学、速度向量和加速度、缓动与弹动、撞球物理、粒子、正向运动学、反向运动学、三维基础等内容。所有实例都是使用原生js开发。示例效果，[看这里](http://www.snsartme.com/animationProgramming/index.html)。
